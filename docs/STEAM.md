# Drift — Steam 页草稿

供 Phase 5 上架前填写 Steamworks 后台时参考。

## 商店短描述

在程序化星野里慢慢漂浮。无任务、无战斗、无压力——只有远方湖面、行星邂逅与极短译文。适合放松、截图与睡前白噪。

## 关于这款游戏

**Drift** 是一款第一人称星际漫游体验。你不是英雄，只是宇宙中的旅人：WASD 平移，QE 升降，不按键也会缓慢向前漂游。

- 程序化星野与柔和 Bloom
- 行星 POI、星门折跃、宇航站遗迹
- 可选微光收集、氛围音乐与浏览器旁白
- 本地漫游档案、成就与 JSON 导出

## 标签建议

`Relaxing` · `Exploration` · `Atmospheric` · `Walking Simulator` · `Indie` · `Singleplayer`

## 定价思路（草案）

- 首发：低价或「Pay what you want」档（$4.99–$7.99 区间调研）
- 与 Web 免费试玩版并存，Steam 版含离线包与成就 / 云存档（若实现）
