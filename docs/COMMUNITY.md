# Drift — 社区与反馈

## 建议渠道

| 渠道 | 用途 |
|------|------|
| GitHub Issues | Bug、功能建议、Spec 讨论 |
| Discord（待建） | 玩家截图分享、版本预告 |
| CHANGELOG.md | 每次发版摘要 |

## Discord 服务器结构（草案）

- `#announcements` — 更新与试玩链接
- `#screenshots` — 截图模式作品
- `#feedback` — 手感与性能反馈
- `#off-topic` — 闲聊

## 反馈我们特别在意的

- 是否晕 3D / 是否需要更多减动效选项
- 加载失败（WebGL、代理、CDN）
- 移动端触控与帧率
- 氛围音乐与 TTS 音量平衡
