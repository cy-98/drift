Drift 项目总览

目标：做成一款精美、治愈、受欢迎的第一人称星际漫游作品。现阶段为 Web 原型（src/game.js）。

核心感受：在星海里慢慢漂浮——宁静、茂远、偶有惊喜。

全局 12/89（约 13%）

视觉

星野 + 后处理 + 统一色相

手感

漂浮式飞行，不晕头

受众

短视频可演示、零门槛试玩

定位、视觉、手感、世界与叙事概念
